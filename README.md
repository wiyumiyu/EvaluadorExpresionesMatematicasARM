# Práctica Programada 1: Evaluador de Expresiones Matemáticas en Ensamblador ARM

Curso: Arquitectura de Computadoras (IC-3101)
Universidad: Tecnológico de Costa Rica (TEC)
Periodo: I Semestre 2024

Descripción:
Desarrollo de una calculadora de expresiones matemáticas implementada en lenguaje ensamblador ARM AA32 sobre Raspberry Pi 4. El proyecto permitió ingresar expresiones aritméticas con operaciones básicas, variables y paréntesis, aplicando validaciones sintácticas, manejo de prioridades operacionales y resolución paso a paso de operaciones matemáticas. Además, se implementaron subrutinas para conversión de datos, validación de entradas y manipulación de memoria a bajo nivel.
